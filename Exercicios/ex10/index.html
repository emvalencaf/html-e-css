Link

Usandos link externos

O link externos são usados para dá acesso a páginas externas ao sites. Por exemplo: clique aqui para ir ao YouTube. 


    <a href="https://wwww.(site).com.br" target="_blank" rel="external">Texto</a>

É sempre recomendável quando for colocar links externos que se coloque a propriedade target="_blank" e rel="external" para manter na navegação o seu site. Dessa forma, evitando que o cliente clique no link e nunca mais volte ao seu site.

Às vezes, quando não se quer associar o site ao site que estar se linkando pode-se atribuir ao atributo rel o "nofollow". O efeito prático é em relação aos mecanismos de buscas que não irão associar o link externo ao site. Isto é usado principalmente quando o dono do site não dá aval ao site linkado. 


    <a href="https://wwww.(site).com.br" target="_blank" rel="nofollow">Texto</a>

Usando link internos

Documento em uma mesma página

O link interno faz uma relação com outra página que está dentro do mesmo servidor do site e em uma relação interna. É sempre importante acrescer ao target o atributo "_self" isso impedirá que se abram várias abas do mesmo site. Nota-se que o default é o target="_self". 


    <a href="nome_.html" rel="next" target="_self">Clique aqui</a> para acessar a 2ª página
Perceba que se o arquivo html da 2ª página estiver na mesma pasta do primeiro arquivo html não será necessário colocar toda a url. Ao atribuir o rel o atributo next estar-se determinando que a outra página html é a seguinte a ele. Por exemplo: Clique aqui para acessar a página 2 do ex10

Documento em uma sub-pasta em relação ao arquivo html

Para link interno a um arquivo que encontra-se em uma sub-pasta em relação ao arquivo deve-se fazer da seguinte forma: 


    <a href="subpasta/nome_.html" rel="next" target="_self">Clique aqui</a> para acessar a 2ª página
Por exemplo: Clique aqui para ir a página 03.

Documento em uma pasta anterior em relação ao html

Para linkar o arquivo html a um arquivo html que encontra-se numa página anterior deve-se atribuir ao href acrescer "../" até aparecer o documento html que deseja se linkar em uma página anterior 


    <a href="../nome.html" rel="next" target="_self">Clique aqui</a> para acessar a 2ª página

link para download

Para baixar um arquivo usando um link interno atribuir ao href o nome do arquvio e colocar na propriedade download o nome do arquivo. 


    <a href="livro/meulivro.pdf" download="meulivro.pdf"> Clique aqui </a> para baixar o pdf.
Às vezes, o navegador não irá reconhecer para fazer download apenas com isso, por essa razão, pode-se acrescer a propriedade type atribuição application/ a extensão do arquivo. Para saber se a extensão é cabível basta acessar esse site. 

    <a href="livro/meulivro.pdf" download="meulivro.pdf" type="application/pdf"> Clique aqui </a> para baixar o pdf.
