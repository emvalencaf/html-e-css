Página 03 do ex10

Linkando um link interno de arquivo que encontra-se em uma sub-pasta em relação ao documento principal. Clique aqui para voltar.

Para voltar a página principal é necessário colocar na tag <a> no atributo href="../index.html". O "../" é usado para usa um arquivo que se encontra numa pasta anterior a que está sendo usada. 


<a href="../index.html" rel="prev">Clique aqui</a> para voltar para a página principal.
