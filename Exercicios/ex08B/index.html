Codigo-fonte

O comando document.getlemntById('teste') é escrito em linguagem JavaScript. A exibição acima deixa um tanto confuso porque a fonte não é monospace. Para isso coloca-se <code>, fica da seguinte forma:
document.getlemntById('teste')
Exemplo de uma código de linguagem python:
num = int(input('Digite um número')) if num % 2 == 0 print(f'O número {num} é par') else print(f'O número {num}é ímpar')
Observa-se que a tag <code> apenas troca a fonte para uma fonte com monospace. A indentação não é colocado para isso e necessário envolver a tag <code> com a tag <pre> 


num = int(input('Digite um número'))
if num % 2 == 0
    print(f'O número {num} é par')       
else
    print(f'O número {num}é ímpar')
Que irá preservar a indentação do código
