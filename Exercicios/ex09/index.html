Trabalhando com Listas

Listas Ordenadas

As listas ordenadas (<ol>) são usadas para ordenar os itens (<li>) e a tag das listas ordenadas(<ol>) é possível atribuir a propriedade type 3 tipos de listas: numérico, letra e algorismos romanos. 

<ol type="(1/a/A/I/i)">
É possível também escolher em qual número/algorismo romano/número de letra a lista vai começar ao atribuir a propriedade start um número: 
<ol type="(1/a/A/I/i) start="(número)"">
Por exemplo: Lista Ordenada por letras que começa pela 4ª letra:

  1. Acordar
  2. Tomar café
  3. Escoar os dentes
  4. Arrumar para o trabalho

Listas não-Ordenadas

As listas não-ordenadas(<ul>) são aquelas em que a ordem dos itens não influencia a lista. Também é possível atribuir a propriedade type da tag (<ul>) qual é o tipo do marcador dos itens (<li>) que podem ser: disk, circle ou square. 

<ul type="(circle/square/disk)">
Por exemplo: Lista não-Ordenada do tipo square:

  • Pão
  • Lite
  • Café
  • Manteiga

EXERCÍCIO 09/B

Juntando listas:

É possível aninhar diferentes tipos de listas.Usando o seguinte expressão: 


    <h4>Minhas matérias favoritas</h4>
    <ol type="1">
        <li>Direito Empresarial</li>
        <ul type="circle">
            <li>Contratos</li>
            <li>Societário</li>
        </ul>
        <li>Direito Civil</li>
    </ol>

Minhas materias favoritas

Lista de Definição

O último tipo de lista possível de se construir usando HTML5 é a lista de definição. Para criar uma lista de definição usa-se a tag<dt> que marca qual será o termo e depois a tag <dd> que marca qual é a definição do termo. Conforme segue a expressão 


    <dt>HTML</dt>
    <dd>Linguagem de marcação para a criação do conteúdo de um site</dd>

Exemplos de Definição

HTML5
Linguagem de marcação para a criação de conteúdo de um site
CCS
Linguagem de marcação para a criação de design de um site
JavaScript
Linguagem de programação para a criação de interatividade de um site
