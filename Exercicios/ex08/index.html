Principais formatações

Negrito/Destaque

Há duas formas para negritar um texto:
1ª - <b> (tag não semântica, é uma tag que só tem forma);
2ª - <strong> (uma tag semântica).

Nesta frase tem-se o termo negritado por <strong> e por <bold>

Itálico/Ênfase

Da mesma forma, há duas formas para tornar um trecho em itálico:
1ª- <em>(Tag semânntica);
2ª-<i>(tag de forma);

Nesta frase tem-se o termo itálico por <em> e por <i>

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

É possível alterar a cor da marcação? Sim, é! Mas não é usando HTML5.

Para alterar a cor da marcação é preciso usar o CSS, uma linguagem de estilo (HTML5 é uma linguagem de significado), devendo-se acrescer ao <mark style:"background-color:(nome da cor)">

Por exemplo: <mark style:"background-color:crimson">.

Texto gande/texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Observa-se que a tag <big> está obsoleta, apesar de ser uma tag semântica. Curiosamente, a tag <small> não está obsoleta.

É possível cogitar que a tag <big> tornou-se obsoleta porque é mais uma formatação do que uma semântica. Em contraponto, a tag <small> continua relevante haja vista que continua usual, principalmente, em contratos.

Texto excluído

O texto excluído é aquele em que ele existe, contudo, ele não deve ser considerado para marcar um texto como excluído é necessário usar a tag <del>. Exemplo: olá, seres.
É uma tag semântica

Texto inserido

O texto incluído é um texto marcado para dá ênfase que ele foi acrescido em uma nova versão do texto usa-se a tag <ins>. Exemplo: olá, seres.
É uma tag semântica

Texto sublinhado

Também é possível sublinhar um texto com a tag <u>. Nota-se que trata-se de uma tag mais de forma do que de semântica.

Texto sobrescrito

Para escrever coisas do tipo: 22; usa-se a tag <sup>.

Texto subscrito

Para escrever coisas do tipo:H2 O; usa-se a tag <sub>.
